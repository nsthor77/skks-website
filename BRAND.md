## 📐 Layout Discipline (Tesla-inspired, our content)

```
✓  Generous whitespace (≥48px between sections on desktop)
✓  Single accent at a time (don't mix blue + gold in one element)
✓  Cards have BORDER not shadow (subtle 1px #E2E6ED)
✓  Background hierarchy: canvas → paper (card) → mist (hover)
✓  Type hierarchy carries the design (not decoration)
✓  Max content width: 1200px (large screens read comfortably)
✓  Buttons: rounded full pills, single solid fill
```

## 🎭 What we DON'T do

```
✗  Pure white #FFF as page bg (text floats)
✗  Pure black #000 text (too harsh)
✗  Emoji as primary UI (childish for B2B)
✗  Multi-color gradients (visual noise)
✗  Drop shadows on cards (use borders)
✗  Comic-style fonts
✗  Too many feedback colors at once
```

## 💎 Where gold appears (sparingly)

```
- "Most Popular" badge on pricing
- Awards/certifications icons
- Premium tier accent
- Achievement / completion indicators
- Subtle hover highlights
```

Never as primary CTA — that stays blue.
