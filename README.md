# 🌳 SKKS — ระบบจัดการโรงเรียนสวนสนขอนแก่น

ระบบครบวงจรสำหรับ **โรงเรียนสวนสนขอนแก่น (SKKS)** ประกอบด้วยเว็บไซต์ประชาสัมพันธ์, ระบบรับสมัคร, ระบบชำระเงิน, และระบบจัดการสำหรับผู้บริหาร พร้อมความสามารถพิมพ์รายงาน A4 ครบทุกแบบ

## ✨ ฟีเจอร์หลัก

### 🌐 เว็บไซต์ (index.html)
- Hero Section พร้อมโลโก้ SKKS
- หลักสูตร 4 แบบ (Pre-K, K, EP, MEP)
- ส่วนแสดงผล O-NET 20 คน เลื่อนแนวนอน
- ฟอร์มสมัครเรียน 4 ขั้นตอน + ระบบชำระเงิน
- Responsive — รองรับมือถือ

### 📊 Dashboard (pages/dashboard.html)
- Quick Action Shortcuts ไปหน้าต่างๆ
- สถิติภาพรวม 4 ตัว: นักเรียน, ใบสมัคร, รายได้, คะแนน O-NET
- กราฟ 4 ชนิด: Bar+Line, Doughnut, Bar Compare, Doughnut Status
- ตารางใบสมัครล่าสุด + Top 10 นักเรียน O-NET

### 📝 จัดการใบสมัคร (pages/applications.html)
- รายการใบสมัคร 28+ รายการ (mock data)
- กรองตามสถานะ, หลักสูตร, ค้นหาชื่อ
- คลิกเข้าดู **ใบสมัครรายบุคคล A4** พร้อมพิมพ์
- ใบสมัครครบทุกหัวข้อ: ข้อมูลนักเรียน, ผู้ปกครอง, ที่อยู่, การชำระ
- ส่วนเซ็นรับรอง พร้อมตราเอกสาร
- Export CSV

### 👨‍🎓 จัดการนักเรียน (pages/students.html)
- 25 ห้องเรียน × 524 นักเรียน (mock data)
- กรองตามหลักสูตร, ระดับชั้น, ค้นหาชื่อ
- **พิมพ์รายชื่อแต่ละห้องเป็น A4** (กระดาษหนึ่งแผ่นต่อห้อง)
- หัวเอกสารแสดงชื่อโรงเรียน, ครูประจำชั้น, จำนวนนักเรียน
- ตารางรายชื่อพร้อมเบอร์ผู้ปกครอง, สถานะการชำระ
- ส่วนเซ็นรับรอง: ครูประจำชั้น + ผู้อำนวยการ
- Export CSV / JSON

### 💰 รายงานการเงิน (pages/finance.html)
**3 แท็บการใช้งาน:**

#### Tab 1 — สรุป
- การ์ดสถิติ: รายรับ, ชำระแล้ว, รอชำระ, ค้างชำระ
- กราฟรายรับรายเดือน + กราฟสถานะ
- **รายงาน A4 แนวนอน**: สรุปแยกตามระดับชั้น + ช่องทางชำระ

#### Tab 2 — รายการชำระ
- รายการชำระทั้งหมด 80 รายการ
- **A4 แนวนอน**: ตารางรายการชำระพร้อมยอดรวม

#### Tab 3 — ใบเสร็จ
- ใบเสร็จขนาด 80mm (เครื่องพิมพ์ใบเสร็จ) หรือพิมพ์ PDF
- จำนวนเงินเป็นทั้งตัวเลขและคำอ่านภาษาไทย
- พิมพ์แยกแต่ละใบ

### 📄 ศูนย์รวมรายงาน (pages/reports.html)
- รายงานทั้งหมดแบ่งเป็น 4 หมวด:
  - 📚 ฝ่ายวิชาการ (4 รายงาน)
  - 💰 ฝ่ายการเงิน (4 รายงาน)
  - 📝 ฝ่ายรับสมัคร (3 รายงาน)
  - 👔 รายงานผู้บริหาร (3 รายงาน)

## 🖨️ การพิมพ์ A4

ทุกหน้าที่มีปุ่ม **"🖨️ พิมพ์ A4"** สามารถ:
1. คลิกปุ่มเพื่อเปิดหน้าต่างพิมพ์
2. เลือก "Save as PDF" เพื่อบันทึกเป็นไฟล์
3. หรือพิมพ์ไปเครื่องพิมพ์โดยตรง
4. UI (sidebar, toolbar, ปุ่ม) จะถูกซ่อนอัตโนมัติ
5. แสดงเฉพาะกระดาษ A4 ที่จัดรูปแบบสมบูรณ์

## 📥 การ Export ข้อมูล

ทุกหน้ามีปุ่ม Export:
- **CSV** — เปิดใน Excel ได้ทันที (มี BOM รองรับภาษาไทย)
- **JSON** — สำหรับนำไปใช้ในระบบอื่น

## 🚀 วิธีใช้งาน

1. แตกไฟล์ ZIP
2. เปิด `index.html` ในเบราว์เซอร์
3. เข้าหน้า Admin โดยคลิก **"Dashboard"** จาก nav
4. ใช้ Sidebar เพื่อเข้าถึงระบบต่างๆ

### ตัวอย่างการใช้งาน

**👉 พิมพ์รายชื่อนักเรียน ป.6/EP**
1. เข้าหน้า "นักเรียน"
2. กรอง "ระดับชั้น" = ป.6
3. กรอง "หลักสูตร" = EP
4. คลิก **"🖨️ พิมพ์ A4"**

**👉 พิมพ์รายงานการเงินส่งผู้บริหาร**
1. เข้าหน้า "การเงิน"
2. แท็บ "สรุป"
3. คลิก **"🖨️ พิมพ์ A4"** (ได้แนวนอน 1 แผ่น พร้อมตารางและสรุป)

**👉 พิมพ์ใบสมัครรายบุคคล**
1. เข้าหน้า "ใบสมัคร"
2. คลิกที่ใบสมัครที่ต้องการ
3. คลิก **"🖨️ พิมพ์"**

## 🔮 พัฒนาต่อ

ระบบนี้เป็น Frontend-only พร้อม mock data หากต้องการใช้งานจริง ควรเพิ่ม:

1. **Backend API** — Node.js/Python/PHP สำหรับเก็บข้อมูลใน Database
2. **ระบบ Login** — Authentication + ระบบสิทธิ์ (Admin/ครู/เจ้าหน้าที่)
3. **Payment Gateway** — Omise / 2C2P / Stripe
4. **อีเมล/SMS** — ส่งยืนยันสมัครและใบเสร็จ
5. **CMS** — จัดการข่าวสาร/กิจกรรม
6. **Backup** — สำรองข้อมูลอัตโนมัติ
