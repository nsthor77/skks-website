1. เข้าสู่ระบบ & ภาพรวม

การเข้าสู่ระบบ

  1. เปิดเว็บของโรงเรียน เช่น ชื่อโรงเรียน.panyaschoolkit.com (หรือโดเมนที่ตั้งไว้)
  2. กรอกอีเมล + รหัสผ่านที่ได้รับ แล้วกด "เข้าสู่ระบบ"
  3. ระบบจะพาไปหน้าแดชบอร์ดตามสิทธิ์ของคุณ

สิทธิ์ผู้ใช้ (Roles)

  • Owner / ผู้บริหาร — เข้าถึงทุกอย่าง รวมการเงินและการตั้งค่า
  • Staff / ครู — จัดการนักเรียน เช็คชื่อ ใส่เกรด
  • ผู้ปกครอง / นักเรียน — ดูข้อมูลของตัวเอง/บุตรหลาน

เปลี่ยนภาษา ไทย/อังกฤษ

กดปุ่ม EN / TH มุมขวาบนของแต่ละหน้าเพื่อสลับภาษา

เคล็ดลับ: ลืมรหัสผ่าน กด "ลืมรหัสผ่าน" ที่หน้า login ระบบจะส่งลิงก์รีเซ็ตไปที่อีเมล

2. ตั้งค่าระบบ แอดมิน

ตั้งค่าครั้งแรกให้โรงเรียนพร้อมใช้งาน — ใช้ Setup Guide นำทาง

ไปที่เมนู "Setup Guide" จะเห็นขั้นตอนทั้งหมดพร้อมแถบความคืบหน้า ระบบจะติ๊กให้อัตโนมัติเมื่อทำเสร็จแต่ละขั้น

ขั้นตอนรายละเอียด
วิชาเพิ่มวิชาที่สอน (คณิต อังกฤษ วิทย์...)
คาบเรียนกำหนดเวลาคาบเรียน + ช่วงพัก
ระบบเกรดเลือก A-F / 4.0 GPA / ผ่าน-ไม่ผ่าน / 0-100%
นักเรียนนำเข้าจาก Excel
ครูนำเข้าจาก Excel
PromptPayใส่ PromptPay ID สำหรับรับค่าเทอม
LINEเชื่อมแจ้งเตือนผู้ปกครอง
ทีมงานเชิญครู / ธุรการเข้าใช้ระบบ

ข้อมูลโรงเรียน

เมนู Settings → กรอกชื่อ (ไทย/อังกฤษ), ที่อยู่, โลโก้, ชื่อผู้อำนวยการ — ข้อมูลนี้จะปรากฏในใบรายงานและเอกสาร ปพ.

สำคัญ: กรอกชื่อโรงเรียน + ที่อยู่ + ชื่อ ผอ. ให้ครบก่อนออกเอกสาร ปพ. เพราะจะแสดงในหัวเอกสาร

3. นำเข้าข้อมูลจาก Excel แอดมิน

ย้ายข้อมูลนักเรียน/ครูจากไฟล์ Excel เดิม โดยไม่ต้องพิมพ์ใหม่

  1. ไปหน้า "นักเรียน" หรือ "ครู" แล้วกดปุ่ม "Import Excel/CSV"
  2. อัปโหลดไฟล์ Excel เดิม (.xlsx หรือ .csv) ได้เลย ไม่ต้องแปลงไฟล์ก่อน
  3. ระบบจับคู่คอลัมน์อัตโนมัติ รองรับหัวคอลัมน์ภาษาไทย (เช่น ชื่อ นามสกุล ห้อง เพศ) — ตรวจดูแล้วแก้ได้ถ้าไม่ตรง
  4. กด "ดูตัวอย่าง" ระบบแปลงค่าให้: เพศ (ชาย/หญิง เป็น male/female), วันที่ (พ.ศ. เป็น ค.ศ.)
  5. กด "นำเข้า" จะเห็นผล "นำเข้า X / ข้าม Y" พร้อมรายงานแถวที่ผิด
ไฟล์ที่มีนักเรียน 300 คน นำเข้าได้ใน 2 นาที แถวที่ข้อมูลไม่ครบจะถูกข้ามและบอกเหตุผล ไม่ทำให้ทั้งไฟล์ล้มเหลว
ฟิลด์ที่จำเป็น: รหัสนักเรียน, ชื่อ, นามสกุล (ครู: รหัสครู, ชื่อ, นามสกุล) — แถวที่ขาดฟิลด์เหล่านี้จะถูกข้าม

4. จัดการนักเรียน แอดมิน

หน้า "นักเรียน" — เพิ่ม/แก้ไข/ค้นหา/กรองตามห้องและระดับชั้น

  • เพิ่มทีละคน: กด "Add Student" กรอกข้อมูล แล้วบันทึก
  • ฟิลด์เพิ่มเติม: Owner ตั้งฟิลด์เองได้ที่ "Configure Fields" (เช่น เลขพาสปอร์ต, กรุ๊ปเลือด)
  • ดูรายละเอียด: คลิกที่ชื่อนักเรียน เห็นข้อมูลครบ + เกรด + การเข้าเรียน

5. จัดการครู แอดมิน

หน้า "ครู" — จัดการข้อมูลครูและกำหนดวิชาที่สอน

  • เพิ่ม/นำเข้าครู (เหมือนนักเรียน)
  • ระบุครูประจำชั้น, กลุ่มสาระ, วิชาที่สอน
  • ฟิลด์เพิ่มเติมตั้งเองได้ (เช่น ภาษาที่พูดได้, การอบรมพิเศษ)

6. เช็คชื่อ + แจ้ง LINE ครู

เช็คชื่อรายวัน + แจ้งผู้ปกครองอัตโนมัติผ่าน LINE

  1. หน้า "เช็คชื่อ" เลือกห้อง + วันที่
  2. กดปุ่มสถานะของนักเรียนแต่ละคน: มา / ขาด / สาย / ลา / ป่วย
  3. ถ้าเชื่อม LINE ไว้ ระบบจะส่งแจ้งเตือนผู้ปกครองอัตโนมัติเมื่อขาด/สาย
ดูสถิติการมาเรียน + ส่งออก CSV ได้ในหน้าเดียวกัน
LINE จะส่งได้ต่อเมื่อ (1) โรงเรียนเชื่อม LINE OA แล้ว และ (2) ผู้ปกครอง add เพื่อน LINE OA และผูก LINE User ID ในระบบแล้ว

7. เกรด & ใบรายงานผล ครู

ใส่เกรดรายห้อง+วิชา+เทอม แล้วเผยแพร่ให้ผู้ปกครองดู

  1. หน้า "เกรด" เลือกปี + เทอม + ห้อง + วิชา แล้วกด Load
  2. ใส่คะแนน (กลางภาค/ปลายภาค/เก็บ) ระบบคำนวณรวม + เกรดอัตโนมัติ
  3. กด "Save All" เพื่อบันทึก (ยังเป็นฉบับร่าง ผู้ปกครองยังไม่เห็น)
  4. กด "Publish" เพื่อเผยแพร่ ผู้ปกครอง/นักเรียนจึงเห็นเกรด

ตั้งระบบเกรด: Owner ตั้งที่ "Grade Scale" เลือก A-F, 4.0 GPA, ผ่าน/ไม่ผ่าน หรือ 0-100%

8. ตารางเรียน แอดมิน

สร้างตารางเรียนรายห้อง รองรับครูสอนคู่ (co-teaching)

  1. หน้า "Schedule" ครั้งแรกกด "Manage Periods" หรือ "ใช้เทมเพลตเริ่มต้น" (8 คาบ)
  2. เลือกปี + เทอม + ห้อง แล้วกด Load
  3. คลิกช่องว่าง เลือกวิชา + ครู (เพิ่มได้หลายคน เช่น ครูต่างชาติ 1 + ครูไทย 2) + ห้อง
  4. ระบบเตือนทันทีถ้าครูซ้อนคาบ (ครูคนเดียวอยู่ 2 ห้องเวลาเดียวกัน)
  • นักเรียนดูตาราง: หน้า My Schedule มีการ์ด "คาบถัดไป" แบบ real-time
  • ครูดูตารางสอน: หน้า Teacher Schedule + ดาวน์โหลด PDF

9. ค่าเทอม & PromptPay แอดมิน

ออกบิล → ผู้ปกครองสแกนจ่าย → ครูอนุมัติสลิป → บันทึกบัญชีอัตโนมัติ

ตั้งค่าครั้งแรก

หน้า Payment Settings → ใส่ PromptPay ID (เบอร์ 10 หลัก หรือเลขผู้เสียภาษี 13 หลัก) + ชื่อบัญชี → ระบบจะสร้าง QR ที่ฝังยอดเงินอัตโนมัติให้แต่ละบิล

ขั้นตอนเก็บเงิน

  1. ออกบิล: หน้า Create Bills สร้างทีละคน หรือทั้งห้องพร้อมกัน
  2. ผู้ปกครองจ่าย: เปิดบิล สแกน QR (ยอดขึ้นอัตโนมัติ) โอน แล้วอัปโหลดสลิป
  3. ครูอนุมัติ: หน้า Payment Approvals ตรวจสลิป อนุมัติ บิลเป็น "ชำระแล้ว" + ลงบัญชีอัตโนมัติ
  4. ดูบัญชี: หน้า Finance รายรับรายจ่ายทั้งหมด
PromptPay ไม่มีค่าธรรมเนียม โรงเรียนได้เงินเต็มจำนวน (ต่างจากบัตรเครดิตที่หักประมาณ 3.65%)

10. เอกสาร ปพ. (1/5/6) แอดมิน

ออกเอกสารทางการตามมาตรฐานกระทรวงศึกษาธิการ

  • ปพ.1 — ระเบียนแสดงผลการเรียน (transcript รวมทุกปี) ที่หน้าเกรดของนักเรียน
  • ปพ.5 — บันทึกผลการเรียนประจำรายวิชา (รายชื่อทั้งห้อง) ที่หน้าเกรด (staff)
  • ปพ.6 — รายงานพัฒนาผู้เรียนรายบุคคล (เกรด+การมาเรียน+คุณลักษณะ) ที่หน้าเกรดของนักเรียน
เลขที่เอกสารออกอัตโนมัติ ตรวจชื่อโรงเรียน/ผอ./ที่อยู่ในหน้า Settings ก่อนออกเอกสาร

11. รายงานภาพรวม (Analytics) ผู้บริหาร

หน้า Analytics สรุปภาพรวมโรงเรียนแบบ real-time

  • จำนวนนักเรียน, ใบสมัครรอ, บิลค้าง, รายได้เดือนนี้
  • กราฟ: นักเรียนตามโปรแกรม, การมาเรียน 7 วัน, การกระจายเกรด, GPA รายห้อง, รายได้ 6 เดือน
  • AI Insights: เตือนบิลเกินกำหนด, นักเรียนขาดเรียน, เกรด F

12. คู่มือผู้ปกครอง / นักเรียน ผู้ปกครอง

เข้าสู่ระบบ

เปิดเว็บโรงเรียน login ด้วยบัญชีที่โรงเรียนให้ จะเห็นหน้าหลักของบุตรหลาน

ดูข้อมูล

  • เกรด — ดูเกรดแต่ละวิชา + GPA + ดาวน์โหลดใบรายงาน/ปพ.
  • ตารางเรียน — ตารางของห้อง + คาบถัดไป
  • การเข้าเรียน — ประวัติมา/ขาด/สาย

จ่ายค่าเทอม

  1. หน้า "บิล/ค่าเทอม" กดบิลที่ต้องจ่าย
  2. สแกน QR ด้วยแอปธนาคาร ยอดเงินขึ้นอัตโนมัติ ไม่ต้องพิมพ์เอง
  3. โอนเงิน แล้วอัปโหลดสลิป รอครูตรวจสอบ
  4. เมื่ออนุมัติ บิลจะเปลี่ยนเป็น "ชำระแล้ว"

13. คำถามที่พบบ่อย (FAQ)

นำเข้านักเรียนแล้วบางแถวหาย?

แถวที่ขาดฟิลด์จำเป็น (รหัส/ชื่อ/นามสกุล) หรือรหัสซ้ำ จะถูกข้าม ดูรายงานหลังนำเข้าว่าแถวไหนผิดเพราะอะไร แล้วแก้ไฟล์ Excel แล้วนำเข้าใหม่เฉพาะแถวนั้น

ผู้ปกครองไม่ได้รับแจ้งเตือน LINE?

ตรวจว่า (1) โรงเรียนเชื่อม LINE OA แล้ว (2) ผู้ปกครอง add เพื่อน LINE OA และผูก LINE User ID ในระบบแล้ว

เปลี่ยนระบบเกรดได้ไหม?

ได้ Owner ไปที่ Grade Scale เลือก/สร้างระบบเกรดที่ต้องการ

QR ค่าเทอมไม่ขึ้นยอด?

ตรวจว่าใส่ PromptPay ID ในหน้า Payment Settings แล้ว (เบอร์ 10 หลัก หรือเลขผู้เสียภาษี 13 หลัก)

ใช้บนมือถือได้ไหม?

ได้ทุกหน้า รองรับมือถือ/แท็บเล็ต ไม่ต้องลงแอป เปิดผ่านเบราว์เซอร์ได้เลย
