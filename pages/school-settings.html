🏫 ข้อมูลโรงเรียน

ℹ️ ข้อมูลนี้จะแสดงบน เว็บไซต์โรงเรียน ใบประกาศนียบัตร และเอกสาร ปพ. โดยอัตโนมัติ

👤 ข้อมูลผู้อำนวยการ

ℹ️ ชื่อและตำแหน่งนี้จะใช้ลงนามบน ใบประกาศนียบัตร และเอกสารทางการ

📱 LINE แจ้งเตือนผู้ปกครอง

ℹ️ ใช้ LINE Official Account (LINE OA) ของโรงเรียนเอง — แพ็กฟรีของ LINE ส่งได้ประมาณ 300 ข้อความ/เดือน (ข้อความตอบกลับอัตโนมัติ เช่น ตอนผู้ปกครองผูกบัญชี ไม่นับ) โควต้าเป็นของบัญชี LINE โรงเรียน

เปิดใช้งาน LINE แจ้งเตือน

เมื่อเปิดใช้งานและใส่ Token แล้ว ระบบจะส่งแจ้งเตือนให้ผู้ปกครองที่ผูก LINE ไว้โดยอัตโนมัติ
👨‍👩‍👧 ผูก LINE แล้ว — จาก — คน

แจ้งเตือนอัตโนมัติ — เลือกได้ว่าจะส่งอะไรบ้าง

📖 วิธีตั้งค่า (6 ขั้นตอน)

  1. สร้าง LINE Official Account ของโรงเรียน (ฟรี) ที่ manager.line.biz
  2. เปิดใช้ Messaging API ในหน้า Settings ของ OA (ระบบจะพาไปเชื่อมกับ LINE Developers)
  3. คัดลอก Channel access token และ Channel secret มาวางในช่องด้านบน แล้วกดบันทึก
  4. นำ Webhook URL (ปุ่มคัดลอกด้านบน) ไปวางใน LINE Developers แล้วเปิด Use webhook
  5. ปิด Auto-reply (ตอบกลับอัตโนมัติ) ใน LINE Official Account Manager
  6. ให้ผู้ปกครองแอด LINE โรงเรียน แล้วพิมพ์ เลขประจำตัวนักเรียน เว้นวรรค วันเกิด เช่น 12345 25/10/2015 — ระบบจะผูกบัญชีให้อัตโนมัติ
